# 可以在油猴脚本中使用
## 关于油猴使用
需要将js插件复制，在油猴插件中新建一个脚本，粘贴，保存，刷新即可。
由于我GreasyFork等账号是刚刚注册，不能发布脚本。暂时只能手动下载导入或者复制粘贴导入。
